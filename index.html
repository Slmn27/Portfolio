About me

Halo! Nama saya Salman Ibnu Hisam, dan saya adalah seorang mahasiswa Matematika di Universitas Islam Negeri (UIN) Sunan Kalijaga, Yogyakarta. Saat ini saya sedang menempuh studi di semester 2 dan sangat menikmati perjalanan akademik saya di sini. Sejak kecil, saya selalu tertarik dengan angka, pola, dan logika. Ketertarikan ini membawa saya untuk memilih jurusan Matematika setelah lulus dari MAN. Di UIN Sunan Kalijaga, saya mendapatkan kesempatan untuk mendalami berbagai cabang Matematika, serta mengembangkan keterampilan analitis dan pemecahan masalah.

Selain mendalami Matematika, saya memiliki minat yang besar di bidang programming. Saya percaya bahwa kemampuan programming sangat penting di era digital ini, terutama untuk menerapkan konsep-konsep matematika dalam berbagai aplikasi praktis. Saya telah mempelajari beberapa bahasa pemrograman seperti HTML, CSS, R, dan C, dan saya tertarik untuk terus mengembangkan keterampilan saya di bidang ini. Di luar kegiatan akademik dan programming, saya memiliki beberapa hobi yang saya nikmati, seperti membaca buku, dan berolahraga. Saya percaya bahwa menjaga keseimbangan antara akademik, minat, dan hobi sangat penting untuk kesehatan mental dan fisik.
